# Lib Rec 📚

A school library management iOS app built with Swift and SwiftData.

## Features
- 📖 Book catalog with search and genre filter
- 👤 Student management
- 🔄 Checkout & return tracking
- ⚠️ Overdue alerts dashboard

## Tech Stack
- Swift / SwiftUI
- SwiftData
- iOS 17+

## How to Run
1. Clone the repo
2. Open `Lib Rec.xcodeproj` in Xcode
3. Select iPhone simulator
4. Press Run (⌘R)
